FABLE · Personalized Agent Bandit Learning for Orchestration

FABLE: Personalized Execution Policies for LLM Agents

A lightweight, user-specific policy model that learns how a frozen black-box agent should use memory, acquire information, and present a response.

FABLE updates online from each user's partial feedback while deterministic constraints control mandatory confirmation and safety behavior.
